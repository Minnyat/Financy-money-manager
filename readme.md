Prerequisites
-------------

- [Kivy >= 1.11.1](http://kivy.org/#download)
- [Kivymd](https://pypi.org/project/kivymd/)
- [Kivymd extensions](https://pypi.org/project/kivymd-extensions.sweetalert/)
- [Python >= 3.7.4](https://www.python.org/downloads/)

-------------

Run: 
`python -m App`
